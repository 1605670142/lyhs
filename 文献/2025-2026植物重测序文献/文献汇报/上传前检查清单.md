## ⚠️ 常见问题

### 图片不显示？
- 检查路径：应该是 `./images/图片1.png`
- 确认文件已上传到 `images` 文件夹
- 刷新页面或清除缓存

### 中文乱码？
```bash
git config --global core.quotepath false
```

### 文件太大？
- 单个文件限制：100MB
- 图片总大小：约1.6MB（正常）
