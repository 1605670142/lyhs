## ❓ 常见问题

### 问题1：提示 "remote origin already exists"

**解决**：
```bash

git remote add origin https://github.com/1605670142/lyhs-research.git
```

### 问题2：推送失败 - Authentication failed

**解决**：使用Personal Access Token

1. 访问：https://github.com/settings/tokens
2. 点击 "Generate new token (classic)"
3. 勾选 `repo` 权限
4. 生成并复制token
5. 推送时：
   - Username: `1605670142`
   - Password: 粘贴token（不是QQ密码！）

### 问题3：图片不显示

**检查**：
- 确认 `images` 文件夹已上传
- 路径是否正确：`./images/图片1.png`
