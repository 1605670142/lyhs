# 快速上传指南（5分钟完成）

## 前提条件检查

在PowerShell中运行：
```bash
git --version
```

- ✅ 显示版本号 → 继续下面步骤
- ❌ 提示找不到命令 → 先看 [Git完整教程_从零开始.md](./Git完整教程_从零开始.md)

---

## 步骤1：配置Git（首次使用）

```bash
# 设置用户名（替换为你的GitHub用户名）
git config --global user.name "你的用户名"

# 设置邮箱（替换为你的GitHub邮箱）
git config --global user.email "your.email@example.com"

# 配置中文显示
git config --global core.quotepath false
```

---

## 步骤2：进入项目目录

```bash
cd D:\yjs\lyhs\qunti
```

---

## 步骤3：初始化Git仓库

```bash
# 初始化
git init

# 添加远程仓库（替换为你的仓库地址）
git remote add origin https://github.com/你的用户名/你的仓库名.git
```

**获取仓库地址**：
1. 登录GitHub
2. 创建新仓库或打开已有仓库
3. 点击绿色 "Code" 按钮
4. 复制HTTPS地址

---

## 步骤4：添加文件

```bash
# 只添加文献汇报文件夹
git add "文献/2025-2026植物重测序文献/文献汇报/"

# 或者添加所有文件
# git add .
```

---

## 步骤5：提交更改

```bash
git commit -m "添加Dipteronia活化石文献汇报"
```

---

## 步骤6：推送到GitHub

```bash
# 推送到main分支
git push -u origin main

# 如果提示分支是master，使用：
# git push -u origin master
```

**首次推送会要求登录**：
- 选择 "Sign in with your browser"
- 在浏览器中登录GitHub
- 授权后自动继续

---

## 步骤7：验证

访问你的GitHub仓库：
```
https://github.com/你的用户名/你的仓库名
```

检查：
- ✅ 文件已上传
- ✅ 图片正常显示
